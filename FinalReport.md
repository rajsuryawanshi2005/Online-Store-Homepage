📝 Project Description

BuyNest Store is a dynamic and responsive online store homepage developed to display categorized products in an interactive and user-friendly manner. The project focuses on modern UI/UX design, real-time data handling using API integration, and smooth user interactions such as filtering, sorting, and cart management.

🚀 Features Implemented
📦 Product listing using API integration
🗂️ Category-based filtering (Electronics, Fashion, Books)
💰 Price sorting (Low to High / High to Low)
🔍 Search functionality for quick product lookup
⭐ Rating filter (3★ & above, 4★ & above)
🛒 Add to Cart functionality with item count
📱 Fully responsive design (mobile, tablet, desktop)
🎨 Modern UI with clean layout and hover effects

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Fake Store API (for dynamic product data)
GitHub (version control)

🎯 Work Summary (Day-wise)
Day 1: Created GitHub repository and structured the project files
Day 2: Designed UI and implemented dynamic product cards using API
Day 3: Added cart system and rating filter, improved interactivity
Day 4: Finalized UI, fixed bugs, optimized code, and tested responsiveness

✅ Outcome

The project successfully meets all the requirements by delivering a clean, responsive, and interactive online store homepage. It demonstrates strong frontend development skills, API integration, and modern UI design practices, making it suitable for portfolio and resume purposes.
